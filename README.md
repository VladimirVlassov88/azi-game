# Azi Game

MVP браузерной карточной игры Azi.

## Что есть сейчас
- HTML, CSS и JavaScript в одном файле
- базовая логика игры
- отображение рук игроков
- розыгрыш карт
- определение победителя взятки
- локальный запуск через Live Server

## Как запустить
1. Открыть проект в Cursor или VS Code
2. Установить расширение Live Server
3. Запустить `index.html` через Live Server

## Статус
Проект в стадии MVP и доработки интерфейса.
